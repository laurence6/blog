我是 Liu，以前在 QQ 空间上发一些文章，但是那里的限制太大了，所以我就申请了自己的域名，搭建了这个博客。

2013 年暑假里我搭建了这个博客。我会在这里发布一些原创、转载、闲谈，基本上是最近碰到什么就写什么。也欢迎大家常来看看，提出你的意见。你可以使用文章末尾的分享按钮将文章分享给你的小伙伴们或者在文末留下你的评论。我很希望各位能够与大家分享你的见闻或者你的心得，如果你想在这里发布你的内容，你可以把它发送到我的邮箱。谢谢大家的支持～

这个博客使用基于 jQuery 的 Bootstrap 框架在 GitHub 搭建，使用了 .tk 的域名，使用 dnspod 的 dns 服务，还使用了 FontAwesome 的图标，以及 百度 的分享功能和 多说 的评论功能，并使用 新浪微博的微相册 作为本站的图床。这是一个开源的博客，你可以访问我的 GitHub 页面来查看这个博客的源代码。

我希望你能够用现代化的浏览器来访问本站，比如 Google Chrome, Mozilla Firefox 等。

你可以点击每个页面左边以及右下角的 RSS 图标来订阅我的博客。
